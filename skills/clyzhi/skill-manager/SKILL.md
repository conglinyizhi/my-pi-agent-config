# 技能管理器

## 触发条件

- 用户要求导入/添加一个新技能（给了一个 GitHub URL）
- 用户要求从某个技能仓库更新已导入的技能
- 用户要求拆分一个多技能聚合仓库

## 执行

按顺序读取并执行以下参考文档：

1. **读取 `references/repo-structure.md`** — 了解本地 `_repo/` 目录结构、软链接约定
2. **读取 `references/import-guide.md`** — 按四步走：分析仓库 → 确认范围 → 执行导入 → 更新 repo.toml
3. 完成后，用一句话汇总：导入了哪些技能、来自哪个仓库
